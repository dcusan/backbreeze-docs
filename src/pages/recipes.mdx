## Overwrite a utility class name

To overwrite a classname change the `name` key in the property config variable:

```scss
// in your project variables file

$bb-text-align-config: (
  'name': 'text-align',
  // ...,,,
);
```

Will yield:

```css
.text-align-center {
  text-align: center:
}
```
